Projetos Acadêmicos

Projeto análise de dados eleitorais

O objetivo deste projeto era proporcionar ao cliente um site de análise de dados com a exibição de estatísticas do eleitorado localizado dentro de sua cobertura no Estado de São Paulo, podendo obter diversas características, como gráficos e tabelas de estado civil, escolaridade e faixa etária, além de informações sobre renda e evolução na quantidade de eleitores, tanto no âmbito municipal (de cada cidade) quanto macro-regiões, como o Vale do Paraíba.

Minhas contribuições pessoais foram em relação ao planejamento do escopo, design de alguns wireframes e principalmente na criação de algumas filtragens de dados utilizando a lib pandas.

Projeto análise de imagens satélite

Esse é um sistema elaborado para permitir a visualização de imagens de satélites disponibilizadas por uma API pública. O sistema permite a troca de satélites bem como o uso de diferentes filtros, download das imagens e a utilização de tiles dinâmicos, também é possível exibir as imagens em suas cores naturais. A imagens são buscadas após a definição de uma área de busca, em que se é escolhida a área de interesse, a data e os satélites desejados.

Minhas contribuições pessoais foram principalmente no planejamento de requisitos do projeto e na construção de rotas no backend.
